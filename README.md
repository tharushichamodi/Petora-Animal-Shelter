# 🐾 Petora Animal Shelter Management System

A full-stack web application developed to simplify animal shelter management by providing an efficient platform for pet adoption, animal records, user management, and shelter administration.

## 📌 Project Overview

Petora is a modern animal shelter management system designed to help shelters manage animal profiles, adoption requests, user accounts, and administrative tasks through an easy-to-use web interface.

This project was developed as an academic full-stack application using the MERN technology stack.

---

## ✨ Features

- 🐶 Animal Profile Management
- ❤️ Pet Adoption System
- 👤 User Authentication & Management
- 📅 Appointment Scheduling
- 🏥 Shelter Administration
- 📱 Responsive User Interface
- 🔍 Search & Filter Animals
- 📂 Image Upload Support

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/tharushichamodi/Petora-Animal-Shelter.git
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm start
```
